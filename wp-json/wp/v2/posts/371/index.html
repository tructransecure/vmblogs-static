Quest has released a new tool called the vWorkspace Desktop Optimizer. The tool contains Quests knowledge on how to optimize a Virtual Desktop, and although it is especially written for vWorkspace its also applicable to Citrix XenDesktop and VMware View.<\/p>\n

The tool contains a list of 40 optimization settings stored in an XML which can be enabled or disabled using either a GUI interface or using the command line.<\/p>\n

Quests own tests using the tools showed a 90% IOPS optimization, 43% less disk space used and RAM savings up to 10% compared to a out of the box Windows 7 Enterprise installation.<\/p>\n
