B\u1ea3o m\u1eadt SSH<\/strong><\/span><\/p>\n

L\u00e0m vi\u1ec7c v\u1edbi server linux l\u1ebd d\u0129 nhi\u00ean b\u1ea1n ph\u1ea3i thao t\u00e1c qua d\u00f2ng l\u1ec7nh th\u00f4ng qua SSH (Secure shell). C\u00e1i t\u00ean n\u00f3 \u0111\u00e3 bao g\u1ed3m ch\u1eef secure – b\u1ea3o m\u1eadt tuy nhi\u00ean \u0111\u1ec3 c\u00f3 \u0111\u01b0\u1ee3c b\u1ea3o m\u1eadt b\u1ea1n ph\u1ea3i c\u1ea5u h\u00ecnh, t\u00f9y ch\u1ec9nh l\u1ea1i, m\u1eb7c \u0111\u1ecbnh th\u00ec kh\u00f4ng b\u1ea3o m\u1eadt t\u00ed n\u00e0o.<\/p>\n

Tr\u01b0\u1edbc khi \u0111\u1ebfn v\u1edbi ph\u1ea7n t\u00f9y ch\u1ec9nh SSH m\u00ecnh s\u1ebd h\u01b0\u1edbng d\u1eabn tr\u01b0\u1edbc c\u00e1c b\u01b0\u1edbc th\u1ee7 t\u1ee5c.<\/p>\n

\u0110\u1ed5i m\u1eadt kh\u1ea9u root:<\/strong><\/p>\n

Khi c\u00e1c b\u1ea1n mua VPS th\u00ec c\u00e1c h\u00e3ng s\u1ebd g\u1edfi email ch\u1ee9a t\u00e0i kho\u1ea3n \u0111\u0103ng nh\u1eadp cho VPS, th\u01b0\u1eddng l\u00e0 root v\u00e0 m\u1ed9t m\u1eadt kh\u1ea9u \u0111\u01b0\u1ee3c t\u1ea1o s\u1eb5n. T\u1ed1t nh\u1ea5t l\u00e0 b\u1ea1n n\u00ean \u0111\u1ed5i m\u1eadt kh\u1ea9u l\u1ea1i sao cho v\u1eeba \u0111\u1ea3m b\u1ea3o d\u1ec5 nh\u1edb v\u00e0 m\u1ee9c \u0111\u1ed9 an to\u00e0n cao so v\u1edbi d\u00e3y k\u00fd t\u1ef1 ch\u1eb3ng c\u00f3 \u00fd ngh\u0129a g\u00ec.<\/p>\n

M\u1ed9t m\u1eadt kh\u1ea9u an to\u00e0n ch\u1ee9a nhi\u1ec1u (t\u1ed1t nh\u1ea5t l\u00e0 d\u00e0i h\u01a1n 10) k\u00fd t\u1ef1 bao g\u1ed3m ch\u1eef in hoa, ch\u1eef th\u01b0\u1eddng, s\u1ed1 v\u00e0 k\u00fd t\u1ef1 \u0111\u1eb7c bi\u1ec7t.<\/p>\n

T\u1ea1o m\u1eadt kh\u1ea9u an to\u00e0n<\/a><\/p>\n

\u0110\u1ec3 \u0111\u1ed5i m\u1eadt kh\u1ea9u root b\u1ea1n c\u1ea7n \u0111\u0103ng nh\u1eadp ssh v\u00e0o vps v\u1edbi t\u00e0i kho\u1ea3n v\u00e0 m\u1eadt kh\u1ea9u \u0111\u01b0\u1ee3c c\u1ea5p<\/p>\n

Ch\u1ea1y l\u1ec7nh<\/p>\n

passwd<\/em><\/p>\n

\u0111\u1ec3 ti\u1ebfn h\u00e0nh \u0111\u1ed5i m\u1eadt kh\u1ea9u t\u00e0i kho\u1ea3n root<\/p>\n

<\/a><\/p>\n

VPS m\u1edbi c\u00e0i n\u00ean c\u0169ng ch\u01b0a c\u00f3 c\u00f4ng c\u1ee5 g\u00ec, m\u00ecnh c\u00e0i th\u00eam 2 c\u00f4ng c\u1ee5 th\u01b0\u1eddng d\u00f9ng l\u00e0 nano v\u00e0 wget<\/p>\n

nano gi\u00fap ch\u1ec9nh s\u1eeda t\u1eadp tin, ch\u1ee9c n\u0103ng t\u01b0\u01a1ng t\u1ef1 vi\/vim c\u00f3 s\u1eb5n nh\u01b0ng s\u1eed d\u1ee5ng d\u1ec5 d\u00e0ng h\u01a1n<\/p>\n

wget gi\u00fap t\u1ea3i t\u1eadp tin t\u1eeb Internet v\u1ec1 m\u00e1y.<\/p>\n

G\u00f5 l\u1ec7nh<\/p>\n

yum install nano wget<\/em><\/p>\n

\u0111\u1ec3 ti\u1ebfn h\u00e0nh c\u00e0i \u0111\u1eb7t. \u1ea2nh d\u01b0\u1edbi do m\u00ecnh \u0111\u00e3 c\u00e0i \u0111\u1eb7t s\u1eb5n n\u00ean m\u00e1y b\u00e1o \u0111\u00e3 c\u00e0i.<\/p>\n

<\/a><\/p>\n

\u0110\u1ec3 server v\u1eadn h\u00e0nh \u1ed5n \u0111\u1ecbnh nh\u1ea5t th\u00ec ch\u00fang ta ph\u1ea3i lu\u00f4n lu\u00f4n c\u1eadp nh\u1eadt c\u00e1c g\u00f3i tin m\u1edbi nh\u1ea5t.<\/p>\n

G\u00f5 l\u1ec7nh<\/p>\n

yum update<\/em><\/p>\n

\u0111\u1ec3 ti\u1ebfn h\u00e0nh c\u1eadp nh\u1eadt server<\/p>\n

<\/a><\/p>\n

T\u1ea1o user:<\/strong><\/p>\n

Sau khi \u0111\u00e3 c\u1eadp nh\u1eadt xong ch\u00fang ta ti\u1ebfn h\u00e0nh t\u1ea1o 1 t\u00e0i kho\u1ea3n, t\u00e0i kho\u1ea3n n\u00e0y \u0111\u00f3ng vai tr\u00f2 trung gian \u0111\u1ec3 k\u1ebft n\u1ed1i SSH t\u1edbi VPS linux tr\u01b0\u1edbc khi s\u1eed d\u1ee5ng t\u00e0i kho\u1ea3n root.<\/p>\n

\u0110\u1ec3 t\u1ea1o t\u00e0i kho\u1ea3n g\u00f5 l\u1ec7nh<\/p>\n

useradd taikhoanmuontao<\/em><\/p>\n

trong b\u00e0i m\u00ecnh t\u1ea1o t\u00e0i kho\u1ea3n monitor<\/p>\n

v\u00e0 \u0111\u1ed5i m\u1eadt kh\u1ea9u cho t\u00e0i kho\u1ea3n v\u1eeba t\u1ea1o, g\u00f5 l\u1ec7nh<\/p>\n

passwd taikhoandatao<\/em><\/p>\n

<\/a><\/p>\n

B\u1ea3o m\u1eadt SSH:<\/strong><\/p>\n

Sau khi ho\u00e0n t\u1ea5t c\u00e1c b\u01b0\u1edbc tr\u00ean ch\u00fang ta ti\u1ebfn h\u00e0nh thi\u1ebft l\u1eadp b\u1ea3o m\u1eadt cho SSH<\/p>\n

G\u00f5 l\u1ec7nh<\/p>\n

nano \/etc\/ssh\/sshd_config<\/em><\/p>\n

<\/a><\/p>\n

\u0110\u1ed5i port SSH<\/span><\/p>\n

Port m\u1eb7c \u0111\u1ecbnh c\u1ee7a SSH l\u00e0 22, do l\u00e0 m\u1eb7c \u0111\u1ecbnh n\u00ean hacker s\u1ebd qu\u00e9t d\u1ef1a theo c\u1ed5ng n\u00e0y \u0111\u1ec3 t\u1ea5n c\u00f4ng brute-force, t\u1ea5t nhi\u00ean l\u00e0 h\u1ecd c\u00f3 c\u00f4ng c\u1ee5 l\u00e0m t\u1ef1 \u0111\u1ed9ng, do v\u1eady \u0111\u1ed5i port gi\u00fap gi\u1ea3m nguy c\u01a1 b\u1ecb t\u1ea5n c\u00f4ng r\u1ea5t nhi\u1ec1u.<\/p>\n

Trong \u1ea3nh d\u01b0\u1edbi m\u00ecnh b\u1ecf ch\u1ecdn d\u1ea5u # v\u00e0 s\u1eeda port m\u1eb7c \u0111\u1ecbnh 22 th\u00e0nh port 2200<\/p>\n

<\/a><\/p>\n

ListenAddress<\/span>: N\u1ebfu b\u1ea1n mu\u1ed1n ch\u1ec9 cho ph\u00e9p truy c\u1eadp SSH t\u1eeb m\u1ed9t \u0111\u1ecba ch\u1ec9 IP c\u1ed1 \u0111\u1ecbnh n\u00e0o \u0111\u00f3 th\u00ec c\u00f3 th\u1ec3 thay \u0111\u1ed5i gi\u00e1 tr\u1ecb n\u00e0y.<\/p>\n

<\/a><\/p>\n

Thay \u0111\u1ed5i tham s\u1ed1 SSH<\/p>\n

MaxAuthTries: cho ph\u00e9p t\u1ed1i \u0111a bao nhi\u00eau l\u1ea7n \u0111\u0103ng nh\u1eadp kh\u00f4ng th\u00e0nh c\u00f4ng.
\nMaxSession: cho ph\u00e9p t\u1ed1i \u0111a bao nhi\u00eau phi\u00ean (session) truy nh\u1eadp SSH<\/p>\n

<\/a><\/p>\n

Kh\u00f4ng cho ph\u00e9p t\u00e0i kho\u1ea3n root tr\u1ef1c ti\u1ebfp \u0111\u0103ng nh\u1eadp ssh<\/p>\n

B\u1ecf ch\u1ecdn d\u1ea5u # \u0111\u1ea7u d\u00f2ng v\u00e0 s\u1eeda gi\u00e1 tr\u1ecb PermitRootLogin<\/span> th\u00e0nh no<\/p>\n

<\/a><\/p>\n

B\u1ea1n th\u00eam tham s\u1ed1 AllowUsers<\/span> \u0111\u1ec3 cho ph\u00e9p t\u00e0i kho\u1ea3n n\u00e0o c\u00f3 th\u1ec3 truy c\u1eadp SSH<\/p>\n

Nh\u01b0 \u1ea3nh d\u01b0\u1edbi l\u00e0 cho ph\u00e9p t\u00e0i kho\u1ea3n monitor (\u0111\u00e3 t\u1ea1o \u1edf b\u01b0\u1edbc tr\u00ean) \u0111\u0103ng nh\u1eadp ssh<\/p>\n

<\/a><\/p>\n

Sau khi \u0111\u00e3 ho\u00e0n t\u1ea5t c\u1ea5u h\u00ecnh c\u00e1c tham s\u1ed1 SSH, ch\u1ea1y l\u1ec7nh d\u01b0\u1edbi \u0111\u1ec3 t\u1ea3i c\u00e1c tham s\u1ed1<\/p>\n

service sshd reload<\/em><\/p>\n

<\/a><\/p>\n

M\u1eb7c \u0111\u1ecbnh t\u01b0\u1eddng l\u1eeda (IPTABLES) cho ph\u00e9p ssh theo port 22, tuy nhi\u00ean do ch\u00fang ta \u0111\u00e3 \u0111\u1ed5i port 22 th\u00e0nh port kh\u00e1c, do \u0111\u00f3 c\u1ea7n c\u1ea5u h\u00ecnh t\u01b0\u1eddng l\u1eeda \u0111\u1ec3 cho ph\u00e9p SSH theo port m\u1edbi.<\/p>\n

C\u00e1c b\u1ea1n c\u00f3 th\u1ec3 truy c\u1eadp \u0111\u1ebfn th\u01b0 m\u1ee5c \/etc\/sysconfig<\/p>\n

V\u00e0 ch\u1ec9nh s\u1eeda file iptables<\/p>\n

nano \/etc\/sysconfig\/iptables<\/em><\/p>\n

<\/a><\/p>\n

S\u1eeda port 22 th\u00e0nh port \u0111\u00e3 \u0111\u1ed5i<\/p>\n

Ctrl + X \u0111\u1ec3 tho\u00e1t v\u00e0 ch\u1ecdn Y (Yes) \u0111\u1ec3 l\u01b0u file<\/p>\n

<\/a><\/p>\n

Ch\u1ea1y l\u1ec7nh<\/p>\n

service iptables reload<\/em><\/p>\n

\u0111\u1ec3 t\u1ea3i l\u1ea1i c\u00e1c tham s\u1ed1 m\u1edbi<\/p>\n

v\u00e0 l\u1ec7nh<\/p>\n

service iptables status<\/em><\/p>\n
